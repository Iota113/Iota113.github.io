Welcome

This is not really a portfolio website, but more of a post-whatever-I-want thing I use to practice some HTML and CSS on the side. I am a year 1 Computer Science and Math undergraduate. My interest lie in applying math to CS and potentially game development. This website is not at all done by myself. A huge chunk of the code is written by my best friends ChatGPT and DeepSeek. Nevertheless, I've learnt a thing or two like adding "defer" when importing a JavaScript script so it works proeprly. I also try to implement something new for every page:

  • The Math Page uses Accordion from Bootstrap.
  • Japan 2024 uses Lightbox for the images and Parallex Scrolling for the background.
  • China 2023 / 2024 uses a Masonry layout (like Pinterest).
