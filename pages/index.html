Welcome Everyone

This is not a portfolio website, but more of a post-whatever-I-want thing I use to practice some HTML and CSS on the side. I am a year 1 Computer Science and Math undergraduate at NUS. My interest lie in applying math to CS and potentially game development. This website is not at all done by myself. A huge chunk of the code is written by my best friends ChatGPT and DeepSeek. Nevertheless, I've learnt a thing or two like adding "defer" when importing a JavaScript script so it works properly. I also try to implement something new for every page:

  • Navigation is done using a Hamburger Menu.
  • The Math Page uses Accordion from Bootstrap.
  • Japan 2024 uses Lightbox for the images and Parallex Scrolling for the background.
  • China 2023 / 2024 uses a Masonry layout (like Pinterest).
  • Media (Games and Shows) uses a Filterable Gallery.
  • Media (Music) uses a YouTube player.
Dev Diary 31 Jan 2025: Integrated Music, Games and Shows and Books; Japan 2024 and China 2023 / 2024 into one page each by putting them into distinct sections and navigating them with on the navbar. Also updated navbar to be more responsive.

It went to the extent of me pulling out the old code from Github and do a one-to-one comparison. Turns out what was causing the issue was me enlarging my browswer view to 110% from the default 100%.
